📝 Objetivo

Este projeto tem como objetivo configurar o Grafana para monitoramento, utilizando o NGINX como proxy reverso para facilitar o acesso e proteger a aplicação. Além disso, será configurado um pipeline de CI/CD para automatizar o deploy das configurações e atualizações.
🚀 Tecnologias Utilizadas

    Grafana – Para visualização de métricas e monitoramento
    NGINX – Para configurar o proxy reverso
    Docker – Para containerizar os serviços
    GitHub Actions – Para pipeline de CI/CD
    
    Configurar o arquivo nginx.conf 

    Pipeline CI/CD

    O arquivo ci-cd.yml define o pipeline para:
    ✅ Build e teste das configurações
    ✅ Deploy automático com Docker Compose
    ✅ Status do Projeto
